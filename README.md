# GraphQL Playground
 
This repository is a starting point for working with GraphQL.
It focuses on core GraphQL concepts through simple, hands-on examples.
 
The project will gradually expand to include automation and testing practices.
 
## Goals
- Learn GraphQL fundamentals
- Work with schemas, queries, and mutations
- Build a basic GraphQL API
- Introduce GraphQL automation step by step
 
## Tech Stack
- Node.js
- GraphQL
- GitHub Codespaces
